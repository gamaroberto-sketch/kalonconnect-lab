# Waiting Room Assets

## Music Files
Place relaxing MP3 files here for the waiting room.

### Example files to add:
- `relaxing-1.mp3` - Calm ambient music
- `relaxing-2.mp3` - Nature sounds
- `relaxing-3.mp3` - Soft piano

### Video Files
- `example-video.mp4` - Welcome video example

## Usage
These files will be available in the "Use Example" dropdown in the Waiting Room settings.
